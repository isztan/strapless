strapless

a minimalist html/css framework

Strapless is a simple kit for building html/css templates. It provides a responsive grid and other goodies, bundled into a simple structure. We are the anti-bootstrap.

download here

Philosophy

  • make the framework invisible
  • use as few files as possible
  • name things clearly
  • encapsulate whenever practical
  • optimize for developer understanding

HTML

  • indent each tier of HTML depth (except on wrapper divs)
  • build the page in html5, but use xhtml syntax
  • use the .row/.container pattern for building full-width sections
  • use simple-grid responsive fluid grid, i.e. col-3-12

CSS

  • indent each tier of CSS depth
  • split CSS into reusable boilerplate and custom skin
  • order common css in top-to-bottom, left-to-right sequence
  • use ems for all font sizes, margins and padding
  • use proportional values for line-height, i.e. 1.1
  • use longhand for margin and padding, i.e. 2em 1em 2em 1em
  • use longhand for background settings
  • use “link blocks” to explicitly set all 4 anchor pseudo-classes
  • dont change the browser’s default font size, even on mobile
  • use 800px and 400px break points
  • no loose styles; wrap typography in a .common styleset
  • use utility classes for common actions
  • use progressive enhancement for special effects

Scripts & Fonts

  • put scripts at bottom of body element
  • use icon fonts whenever possible instead of images

PHP

  • indent each tier of code, separate sections with blank lines
  • use curly brace syntax
  • use helper functions and includes when it helps
  • use underscores for all variables, arrays and functions
  • use camelCase for methods and objects. Capitalize classes.

WordPress

  • use as few template files as possible
  • use 1 master loop file, with conditional statements to change its behavior

Files

  • lowercase filenames
  • /styles, /scripts, /images, /fonts
  • no loose images – put all image files in categorized subfolders
  • underscores instead of dashes
  • prefix all partials with an underscore
